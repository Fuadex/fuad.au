# fuad.au — status snapshot (2026-07-18)

Cross-app status as of the full-project audit (2026-07-18). Per-app detail lives in each app's
own docs (`rotation/ROADMAP.md` + `ARCHITECTURE.md`, `culture/docs/AUDIT_2026-07.md`,
`canvas/PLAN.md`). Earlier cross-app snapshot (2026-07-08 Fable-leverage plan) lives in
`notes/STATUS-2026-07.md` (gitignored).

---

## Hub / infrastructure

- **PWA shell shipped** across all three apps. Each ships `manifest.webmanifest`, `sw.js`,
  `icon-192.png`, `icon-512.png` (all in `apps.json` deploy lists). Tiered service-worker
  caching; installable on desktop and mobile. HTTPS-only registration. No Tauri/Capacitor —
  deliberate: PWA + Pages keeps push-to-live instant.
- **Content-hash cache-busting** is now automatic for Rotation and Canvas. `stage-site.js`
  stamps every local unversioned script/CSS ref in both the hub root `index.html` and each
  per-app `index.html` with an 8-char content hash. The hub's own `hub.css` ref is now
  stamped (was missing; caused stale-style deploys).
- **SW cache epoch** (`__BUILD__` token in each `sw.js`) is replaced at stage time with a
  digest of the full staged app directory. Any deployed byte changing opens a fresh epoch.
- **stage-site.js: missing files always fail** — the CI-only guard removed (2026-07-18 audit).
- **sync.yml smoke steps** now carry distinct names: "Smoke test — slug/join invariants"
  and "Smoke test — dataset sanity".
- **OG / Twitter cards and favicons** are present on all three apps. Culture gained a
  `favicon.svg` (previously missing).
- **apps.json**: culture deploy list gained `favicon.svg` + four PWA files; rotation deploy
  list gained `instrumentals.js`; `blurb-demo.js` is removed.
